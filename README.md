## SoundSwap

Aplicación de escritorio para transferir playlists entre YT y Spotify

## ⚙️ Instalacion

1. Clona el repositorio
2. Genera tus credenciales de Spotify/Youtube y guardalos junto al repositorio
3. Ejecuta el script ````main.py````
4. Listo
